Url shortener micro-service

Input

https://infinite-tundra-39395.herokuapp.com/add/http://www.google.com

Output

{"original_url": "http://www.google.com","short_url": "https://infinite-tundra-39395.herokuapp.com/1"}

Usage

Now

  • https://infinite-tundra-39395.herokuapp.com/1
